#Playwright Test Automation Framework

This project is a **Playwright-based Test Automation Framework** built using **TypeScript**.  
It demonstrates end-to-end UI testing with real-world scenarios like form handling, file upload, and table validation.

---

#Features

- Page Object Model (POM) structure
- Cross-browser testing (Chromium, Firefox, WebKit)
- File upload automation
- Static table validation
- Dynamic locators handling
- CI/CD integration using GitHub Actions
- Clean and scalable framework design

---

#Tech Stack

- **Automation Tool**: Playwright
- **Language**: TypeScript
- **Framework**: Node.js
- **CI/CD**: GitHub Actions
